# 🎬 MovieVault | The Royal Experience

**MovieVault** is a sophisticated web application that allows users to explore, search, and manage their favorite movies using the **TMDB API**. Built with a modern tech stack focused on performance and security.

---

## 🚀 Features
- **Live Search:** Find any movie from the massive TMDB database.
- **User Authentication:** Secure Login/Register system using JWT.
- **Personal Vault:** Add movies to your favorites or watchlist (PostgreSQL).
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
- **Admin Dashboard:** specialized views for managing content.

---
