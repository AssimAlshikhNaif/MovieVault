# 🎬 MovieVault - The Royal Cinema Experience

A premium movie gallery and critique platform built with the **PERN Stack** (PostgreSQL, Express, React, Node.js).

## 🚀 Quick Start

### 1. Backend Setup

- Navigate to the root directory.
- Create a `.env` file with:
  `DATABASE_URL=your_postgres_url`
  `JWT_SECRET=your_secret_key`
- Run `npm install` then `npm run dev`.

### 2. Frontend Setup

- Navigate to the frontend folder.
- Run `npm install`.
- Run `npm run dev` to launch the masterpiece.

## 🛠 Tech Stack

- **Frontend:** React, Tailwind CSS, Lucide Icons.
- **Backend:** Node.js, Express.js.
- **Database:** PostgreSQL.
- **Security:** JWT Authentication, Bcrypt Password Hashing.

## 👑 Features

- Real-time Movie Data from TMDB API.
- Royal Membership System (Login/Register).
- Exclusive "Critique" section for movie reviews.
- Cinematic UI with Golden Accents.
